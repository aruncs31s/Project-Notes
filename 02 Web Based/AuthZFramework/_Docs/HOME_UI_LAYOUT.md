## Interactive Elements

### Card Hover Effects
- **Scale**: Slightly increases (scale-105)
- **Shadow**: Increases from sm to md
- **Border**: Changes color to primary accent
- **Transition**: Smooth 300ms easing

### Button States
- **Hover**: Background darkens, text brightens
- **Active**: Scale reduces slightly
- **Disabled**: Opacity reduces to 75%
- **Focus**: Ring outline appears

### Dark Mode Toggle
- **Location**: Header right side
- **Icon**: Moon (light mode) / Sun (dark mode)
- **Storage**: localStorage persistence
- **Auto-detect**: System preference fallback
